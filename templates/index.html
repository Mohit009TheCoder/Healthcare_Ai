ML Architecture — What Changed in v3.0
Symptom Model (FIXED)

Previously used the world disease dataset (wrong data). Now uses DiseaseAndSymptoms.csv (4,920 records, {{ stats.diseases }} diseases, {{ stats.symptoms }} binary symptom features) with a soft-voting ensemble (RandomForest + Naive Bayes + Logistic Regression). Also now shows disease-specific precautions after every diagnosis.

Diabetes Model

Random Forest (200 trees, max_depth=10) on 11 clinical features. StandardScaler preprocessing, 80/20 split. Precautions now shown for positive results.

Drug Recommendation

Content-based filtering using TF-IDF (500 features) on disease+effectiveness+side_effects+risk_level. Rating-weighted ranking. Triggered automatically after symptom diagnosis.
